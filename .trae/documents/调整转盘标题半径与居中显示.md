## 目标
- 将标题距圆心的半径设置为圆半径的 60%，确保处于扇形“安全区域”。
- 标题沿扇形角平分线精准居中显示，既不贴内圆也不顶外缘。

## 技术要点
- 圆半径单位：容器百分比中圆心到外缘为 50（即 100% 宽高的 50%）。因此“圆半径的 60%”= 50 × 0.6 = 30。
- 使用变换链而非手算坐标：`translate(-50%, -50%) rotate(midAngle) translate(0, -rPct%)`，其中 `midAngle = startAngle + segmentAngle / 2`，`rPct = 30`。
- 安全边距：按实际像素计算最小/最大允许半径，避免碰到中心白圆或外缘。

## 代码修改
- 文件：`src/components/Wheel.tsx`
1. 在扇形循环处，替换现有的 `x/y` 极坐标计算为变换链：
   - 现行：使用 `x=50+r*cos(...)`、`y=50+r*sin(...)` 并旋转容器。
   - 改为：容器固定在圆心，样式：
     - `left: '50%'`, `top: '50%'`
     - `transform: translate(-50%, -50%) rotate(midAngle) translate(0, -rPct%)`
2. 计算安全半径 `rPct`：
   - `const wheelSize = 320; // 与 w-80 h-80 对应`
   - `const hub = 56; // 中心白圆直径`
   - `const margin = 10;`
   - `const radiusPx = wheelSize / 2;`
   - `const safeMinPct = ((hub/2 + margin) / radiusPx) * 50;`
   - `const safeMaxPct = 50 - (margin / radiusPx) * 50;`
   - `const desiredPct = 50 * 0.6; // 30`
   - `const rPct = Math.min(safeMaxPct, Math.max(safeMinPct, desiredPct));`
3. 文字样式保持竖排且中心对齐：
   - `display: 'flex', alignItems: 'center', justifyContent: 'center'`
   - `writingMode: 'vertical-rl', textOrientation: 'upright'`
   - `fontSize` 依标题长度自适应（如 `>=5 ? 14 : 16`）。
